# Try Out Pemrograman Web Service And Managment

## Setup 
- Clone This repository
- Copy File `.env` to `.env.development`
- Open `.env.development` and set variabel :
```
DB_CONNECTION=mysqli
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
```
